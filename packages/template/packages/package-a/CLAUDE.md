# CLAUDE.md（package-a）

このパッケージ固有の Claude Code 向け指示。

## ファイル構成

```
src/       ← ソースコード
docs/      ← このパッケージの仕様・セットアップ（変化が遅い情報）
plan.md    ← このパッケージ内の計画（横断の話は in-progress/plan.md へ）
```

## 出力ルール

| 出力の種類 | 書く場所 |
|---|---|
| パッケージ固有の計画 | `in-progress/plan/package-a.md` |
| API仕様 | `docs/api.md` |
| データ構造 | `docs/data.md` |
| ビジネスルール | `docs/rules.md` |
| セットアップ手順 | `docs/setup.md` |
